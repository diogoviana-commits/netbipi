# NetBIPI v1.0.0

First public release of the NetBIPI open source platform.

## Summary

NetBIPI centralizes the daily workflow of N1, N2 and operations teams in a single panel. The platform brings together monitoring, service desk, inventory, network diagnostics, knowledge base, reports and live updates in one place.

## What this release includes

- Real-time dashboard and alert management
- Zabbix webhook integration
- GLPI ticketing integration
- Asset inventory and infrastructure map
- Network diagnostics from the browser
- Reports, logs and shift handoff views
- Kiosk mode for NOC screens
- Docker Compose profiles for modular deployment

## Launch notes

- Sensitive defaults were removed from the public configuration files.
- Demo credentials are kept only for local lab flows.
- The project is ready for GitHub publication, LinkedIn announcement and community contributions.

## Suggested GitHub release description

NetBIPI v1.0.0 is the first public release of the platform.

It provides an open source operational hub for teams that need to centralize alerts, tickets, assets, diagnostics and reporting in one interface. The project is built with React, TypeScript, Node.js, PostgreSQL and Docker, and includes optional integrations with Zabbix and GLPI.

This release is designed for lab usage and as a clean starting point for teams that want to adapt the platform to a real operations environment.
